Provided Structure

In the optical-toys folder, you will find The following files and folders:

  • assets/ - a folder for any other assets you may need (e.g. images)

Create a page called grid.html, inside it create a grid layout. The requirements for this grid are:

  • There should be three columns
  • As many rows as required
  • An image with the class .wide should span 2 columns
  • An image with the class .tall should span 2 rows
  • An image with the class .big should span 2 columns and 2 rows
  • An image with the class .hero should span 3 columns and 2 rows
